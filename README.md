# Asteroids (Pygame)

A modern take on the classic arcade game, built with Python and the Pygame library. This project focuses on object-oriented programming (OOP) principles, collision detection, and game loops.

## Features
- **Vector-based Movement:** Real-time physics for player rotation and thrust.
- **Dynamic Collision System:** Scalable collision detection between asteroids, shots, and the player.
- **Wrap-around Screen:** Classic arcade wrapping logic for all game objects.
- **Recursive Asteroid Splitting:** Larger asteroids break into smaller, faster fragments.

## Tech Stack
- **Language:** Python 3.13
- **Graphics:** Pygame
- **Environment Management:** uv

## Getting Started

### Prerequisites
Ensure you have `uv` installed. If not, you can install by following the instructions at: https://docs.astral.sh/uv/getting-started/installation/

### Installation & Execution

1. Clone the repo
2. Run the game: uv run main.py

### Controls

WASD: Move and Rotate
Spacebar: Fire Cannon
